# Platform Research Workspace

This folder is reserved for platform-level research instead of wallet-level imitation.

Current scope:

- Normalize historical records from `Z` and `P`
- Compare platform signal quality, timing, and category strength
- Study cycle changes and regime shifts

Suggested workflow:

1. Place raw exports under `raw/`
2. Convert them into the canonical schema under `normalized/`
3. Run downstream analysis on the normalized files only
